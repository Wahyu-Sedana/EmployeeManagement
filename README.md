# Employee Management API

A RESTful API built with **Spring Boot** and **MySQL** to manage employees, departments, and positions.  
Includes **validation**, **global exception handling** and **DTO mapping**.

## Features

- CRUD operations for:
  - Employees
  - Departments
  - Positions
- Input validation with custom error messages
- Global exception handling with structured API response
- DTO-based request/response for clean API design
- Swagger UI for interactive API documentation
- UUID as primary keys for all entities
- Hibernate JPA with MySQL database

## Tech Stack

- Java 21
- Spring Boot 4.0
- Spring Data JPA
- Hibernate
- MySQL
- Lombok
- Jakarta Validation
